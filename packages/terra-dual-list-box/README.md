# Terra DualListBox


[![NPM version](https://badgen.net/npm/v/terra-alert)](https://www.npmjs.org/package/terra-dual-list-box)
[![Build Status](https://badgen.net/travis/cerner/terra-core)](https://travis-ci.com/cerner/terra-core)

Terra-Dual-List-Box enables user to transfer the elements between two list box in an intuitive and efficient way.

One or more elements can be selected from either column, to transfer between two listbox. direction button can be used to transfer elements between list. The left list box is considered the source and the right column is considered the target.
